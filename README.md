# Mirai明日方舟掉率查询插件

### 1、简介

查询明日方舟地图及物品的掉落统计，数据来源：[企鹅物流数据统计](https://penguin-stats.io/)

适配mirai1.x版本

### 2、使用

1）下载：https://github.com/aowubulao/arknights-drop-plugin/releases

2）下载后放入plugins文件夹

3）启动mirai-console，启动后请等待插件加载数据，时间大概在20s-70s

4）指令：

- **物品查询#扭转醇**
- **地图查询#1-7**
- 好友消息：**\#刷新掉率数据**
